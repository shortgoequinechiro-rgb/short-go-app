Critical Issues

AI SOAP Note Generation is Broken
/api/generate-soap — Uses client.responses.create which doesn't exist in the OpenAI SDK. Should be client.chat.completions.create. This endpoint will throw a runtime error on every call.
QuickBooks OAuth is Incomplete
/api/quickbooks/auth — References an undefined OAuthClient, missing scope configuration, and no error handling. OAuth flow will fail.
Reports API Has Incomplete Logic
/api/reports — Active patients calculation has a TODO comment with incomplete code (lines 79-85). Hardcoded to 6 months / 20 items with no date range filtering.
Missing Page: /select-mode
The signup flow calls router.push('/select-mode') but no page exists at that route. Users hitting this path will get a 404.
Offline Sync is Read-Only
Dexie DB defines PendingHorse, PendingVisit, PendingIntakeForm, PendingAppointment tables, but the sync-back logic to push pending changes to Supabase isn't fully implemented.
Missing horse_contacts Table
Referenced in the horse detail page UI but no migration or database table exists. Would track farrier, vet, trainer, barn manager contacts per horse.
